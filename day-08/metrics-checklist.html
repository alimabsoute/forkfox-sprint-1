Day 8 · Field Reference

Metrics Checklist (Day 8 + Day 13)

8-source pull list — GA4, Reddit Ads, Meta, TestFlight, more
Header
Use this exact list for every metrics pull.
1. Ga4 (Forkfox.Ai)
Open: https://analytics.google.com → ForkFox property
Pull these dimensions over the last 7 days
or whatever sprint window
- Total users - Total sessions - Engaged sessions - Avg engagement time per session - Users by source / medium (filter to non-bot cities) - Real Philadelphia visitors (filter: City = Philadelphia, exclude Eagleville) - Real Bay Area visitors (filter: City IN [San Mateo, San Jose, San Francisco], exclude founder list) - iOS share of mobile traffic (Tech > Operating system) - Form_submit events count (Events report) - /?signed-up=true page views (Pages and screens report)
Log into tracker → Daily Signups + Decision Log.
2. Reddit Ads Dashboard
Open: https://ads.reddit.com → Campaign view
Pull per ad variant
- Impressions (last 7 days) - Clicks - CTR (click-through rate) - CPC (cost per click) - Total spend
Calculate Cost Per Install
CPI = (total spend) / (TestFlight installs from Reddit ads UTM)
Log in tracker → Ads tab.
3. Meta Ads Manager
Open: https://business.facebook.com/adsmanager → Campaign view
Same pulls as Reddit per ad variant
- Impressions - Link clicks (NOT total clicks — link clicks only) - Cost per link click - CTR (link click) - Frequency (impressions per user — if >4, audience is fatiguing) - Total spend
Pixel events count
last 7 days, Events Manager
- PageView - ViewContent (/beta page hits) - Lead (form_submit) → If Lead events ≥ 50 in past 7 days → conversion-mode unlock available.
Log in tracker → Ads tab.
4. Reddit Organic
For each post
open the post, scroll to top
- Upvotes - Comments - Shares (if visible) - Crossposts
In your tracker → Reddit tab, log the 24h, 48h, 7d snapshots.
GA4 cross-reference
- In GA4 → Acquisition → Sources, look for "reddit.com" referral. - Note clicks per post.
5. Dm Tracker
Open tracker → Outreach tab.
Calculate
- Total DMs sent (cumulative) - Reply rate (responses / total) - Install rate (installs / responses)
Identify top-performing personalization angle (which restaurant references correlated with replies).
6. Apple Testflight Insights
Open: https://appstoreconnect.apple.com → My Apps → ForkFox → TestFlight
Pull
- Total installs (cumulative) - Active users (last 7 days) - Sessions (last 7 days) - Avg session duration
Cross-reference: TestFlight installs vs GA4 form_submit events. The ratio shows actual install conversion.
7. Directory Status Audit
Google search each
"ForkFox betalist", "ForkFox saashub", "ForkFox alternativeto", "ForkFox crunchbase", "ForkFox wellfound", "ForkFox indiehackers"
Mark each as Live or Pending in tracker → Directories tab.
8. Time Log
Estimate hours spent on the sprint this week (Ali + Nik combined). Log in tracker → Decision Log: "Week N hours invested: X." Used for ROI calculation in sprint review.
At The End
Write 1 line in Decision Log
"Week N rollup: $[spend] → [installs] installs → $[CPI] blended CPI. Top channel: [X]. Bottom channel: [Y]. Plan: [Z]."
Done? Mark it complete in your tracker, then go back to the day overview.
